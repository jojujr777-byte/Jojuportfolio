HOW TO ADD YOUR IMAGES
======================

1. PROFILE PHOTO
   - Save your photo as:  profile.jpg
   - Place it in this folder (assets/images/)
   - Recommended size: at least 400x400 px, square crop works best

2. PROJECT THUMBNAIL IMAGES (optional — shown as video poster frames)
   Save these files in this folder (assets/images/):
   - gearup-thumb.jpg        → thumbnail for Gear Up project video
   - farmers-thumb.jpg       → thumbnail for Farmers Portal project video
   - portfolio-thumb.jpg     → thumbnail for Portfolio project video

3. CERTIFICATE IMAGES
   Save your certificate images in the certs/ sub-folder (assets/images/certs/):
   - ibm-data-analytics.jpg      → IBM Data Analytics (Coursera)
   - angular-infonix.jpg         → Angular Front-End (Infonix Solutions)
   - cyber-security-iit.jpg      → Cyber Security & Privacy (IIT Madras / NPTEL)
   - cloud-infosys.jpg           → Cloud Technologies (Infosys Springboard)
   - database-microsoft.jpg      → Database Integration Management (Microsoft / Coursera)

   Supported formats: .jpg  .jpeg  .png  .webp
   Recommended size: at least 800x600 px for good quality in the lightbox viewer.

Once you place the images with these exact filenames, they will automatically
appear in the portfolio without any code changes needed.
